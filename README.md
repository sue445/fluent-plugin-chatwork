|------------------------|---------|------|
| >= 2.0.0 | >= v0.14.0 | >= 2.1 |
|  < 2.0.0 | >= v0.12.0 | >= 1.9 |

## Installation


    $ gem install fluent-plugin-chatwork

## Configure

### For non-buffered

```
<match **>
  @type        chatwork
  api_token    YOUR_SECRET_TOKEN
  room_id      0000000000
  message      Hello ChatWork!\n<%= record["value"] %>
</match>
```

### For buffered

```
<match **>
  @type        chatwork
  api_token    YOUR_SECRET_TOKEN
  room_id      0000000000
  message      Hello ChatWork!\n<%= record["value"] %>
  buffered     true
  <buffer tag>
    @type memory
  </buffer>
</match>
```

* api_token
  * secret api token
* room_id
  * send message to this room
* message
  * message content
  * support erb format
  * support newline character (\n)
* buffered
  * Switch non-buffered/buffered output
